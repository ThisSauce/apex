# APEX — Personal Training Log

## Deploy to Vercel (recommended, free)

1. **Create a GitHub account** at github.com if you don't have one
2. **Create a new repo** at github.com/new — name it `apex`, keep it public
3. **Upload these files** by clicking "uploading an existing file" on the repo page
   - Drag the entire contents of this zip into the uploader
   - Click "Commit changes"
4. **Go to vercel.com** → Sign up with your GitHub account
5. Click **"Add New Project"** → Import your `apex` repo
6. Leave all settings as default → click **"Deploy"**
7. In ~60 seconds you'll get a URL like `apex-yourname.vercel.app`

## Add to iPhone Home Screen
1. Open the URL in Safari on your iPhone
2. Tap the **Share** button (box with arrow)
3. Tap **"Add to Home Screen"**
4. Tap **"Add"** — it'll appear as a full-screen app icon!

## Run locally
```bash
npm install
npm run dev
```
